[DataCite](http://www.datacite.org) is a DOI registration agency for datasets. The DataCite Data source tracks events for DataCite DOIs instead of other works referenced by DataCite.

## Example Response

```json
{
  "responseHeader": {
    "status": 0,
    "QTime": 2
  },
  "response": {
    "numFound": 1,
    "start": 0,
    "docs": [
      {
        "doi": "10.5061/DRYAD.8515",
        "relatedIdentifier": [
          "HasPart:DOI:10.5061/DRYAD.8515/1",
          "HasPart:DOI:10.5061/DRYAD.8515/2",
          "IsReferencedBy:DOI:10.1371/JOURNAL.PPAT.1000446",
          "IsReferencedBy:DOI:"
        ],
        "creator": [
          "Ollomo, Benjamin",
          "Durand, Patrick",
          "Prugnolle, Franck",
          "Douzery, Emmanuel J. P.",
          "Arnathau, Céline",
          "Nkoghe, Dieudonné",
          "Leroy, Eric",
          "Renaud, François"
        ],
        "publisher": "Dryad Digital Repository",
        "title": [
          "Data from: A new malaria agent in African hominids."
        ],
        "publicationYear": "2011"
      }
    ]
  }
}
```
