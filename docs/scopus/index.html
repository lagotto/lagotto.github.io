Scopus

Scopus is an abstract and citation database of peer-reviewed literature. Information about access to the Scopus API can be found here.

ALM Name scopus
ALM Configuration default
ALM Core Attributes id (as doi)
url (as http://dx.doi.org/DOI)
contributor(s)
ALM Other Attributes ISSN
journal title
journal abbreviation
title
volume
issue
first page
year
publication type
citation count
Scopus ID
Protocol REST
Format JSON or XML
Rate-limiting unknown
Authentication API key and institutional token
Restriction by IP Address no
API URL Cited-By https://api.elsevier.com/content/search/index:SCOPUS?query=DOI(DOI)
